Sistemas Sal

Automação de simples nacional

O bot automatiza a obtenção do relatório de clientes no SSW, realiza consultas individuais e compara os dados com a base da Sal. Qualquer alteração no Simples Nacional é imediatamente atualizada no banco de dados da empresa, garantindo conformidade fiscal e evitando multas. Ao final do dia, o sistema gera e envia um relatório em formato .xls para o setor responsável, mantendo a equipe sempre informada.
Tecnologias: Python

Sistema de devolução

O cliente solicita a devolução, e o sistema gera automaticamente a OS no SSW via web scraping, notificando a base e retornando o número do pedido.
Tecnologias: Python, JavaScript, PHP, HTML, CSS

Este sistema permite que o cliente solicite a devolução de forma simples e eficiente. Assim que a solicitação é feita, um web scraping em Python interage com o SSW, gerando automaticamente a Ordem de Serviço (OS). O número da OS é imediatamente retornado ao cliente e, ao mesmo tempo, todas as informações da devolução são enviadas diretamente para a base de dados, garantindo um fluxo ágil e preciso. Isso elimina processos manuais, reduz erros e melhora significativamente a experiência do usuário.

App de Etiquetas

Este sistema monitora os e-mails recebidos de mais de 20 clientes. Sempre que um e-mail contém uma nota fiscal, ele é lido e salvo na API, que retorna as informações para o app. Cada cliente gera uma etiqueta com base nas notas fiscais recebidas. Ao final, é gerado um relatório com os itens lidos e gerados, garantindo a rastreabilidade e evitando extravios para os clientes.
